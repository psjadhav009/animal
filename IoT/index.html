Experiment 1: LED Blinking

Objective

To blink an LED using an Arduino board.

Components Required

Arduino UNO, LED, 220Ω resistor, jumper wires and breadboard.

Arduino Code

void setup() {
    pinMode(13, OUTPUT);
}

void loop() {
    digitalWrite(13, HIGH);
    delay(1000);

    digitalWrite(13, LOW);
    delay(1000);
}
